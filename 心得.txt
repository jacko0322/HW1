這次我以 NBA 球員資料為主題設計網站，並使用開放 API 作為資料來源。一開始因為查詢條件包含了過多球員與賽季，導致抓取到的數據混亂不清。後來我將範圍縮小，聚焦於本賽季媒體評選的 NBA 前 100 名球員，並以此進行資料搜尋。

在資料抓取過程中，部分球員的數據無法正確取得。深入了解後發現，這些錯誤多與球員名稱格式有關，例如外國球員的名字拼法不同、使用縮寫，或是名字過長等問題。針對這些情況，我逐一進行修正，最終順利取得所有球員的資料。

為了提升網站的美觀性，我另外從官方網站手動下載每位球員的照片，並整合進排版設計中。完成後，我將網站部署至 GitHub 進行備份，卻發現 API 無法在 GitHub 上正常運作。進一步查明後得知，GitHub Pages 採用 HTTPS，而我的 API 僅支援 HTTP，導致請求被阻擋。

為了解決這個問題，我採用兩個網頁分別處理的方式：一個網頁負責從 API 取得資料並儲存為本地 JSON 檔案；另一個網頁則讀取這份 JSON 檔案，以顯示資料。透過這種方式，成功克服了 GitHub Pages 的限制，也順利完成整個網站的製作。
